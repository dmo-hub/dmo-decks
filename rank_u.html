U rank Digimons are 1 step above SSS+ and this category is populated by God-like Digimon that are so powerful, they are considered gods. Most notable Digimons of this Rank

  • Alphamon Ouryuken (Extreme)
  • Bloomlordmon
  • Done Devimon
  • Eosmon LV6
  • Gallantmon (Crimson Mode) (Awaken)
  • Goddramon
  • Holydramon (Awaken)
  • Imperialdramon Paladin Mode (Awaken)
  • Kuzuhamon - Miko Mode
  • Last Evolution: Kizuna
  • Lilithmon (X-Antibody) (Awaken)
  • Lucemon: Satan Mode (Extreme)
  • Omegamon - Merciful Mode
  • Shoutmon X7 Superior Mode
  • Susanoomon (Extreme)
  • ZeedMillenniummon (Awaken)

  • Abbadomon Core
